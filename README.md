# Материалы МАИ

Данный репозиторий создан для хранения конспектов с лекций и записей с семинаров Московского
Авиационного Института. В первую очередь здесь будут публиковаться материалы с тех пар, на которые
хожу я (автор репозитория)

## Как скачать?

Наиболее интересен вопрос о скачивании готовых PDF файлов. Они представлены в странице [releases](https://github.com/ognevny/MAI-materials/releases/). На самой странице доступны ссылки для скачивания, отсортированные по семестрам. Скачивание автоматически произойдёт при нажатии на ссылку.

## Как читать?

1. Файлы PDF можно читать в любой удобной смотрилке. Формат автоматически подобран такой, что он готов к печати на листах А4

2. Исходники в Markdown могут быть открыты непосредственно в GitHub

## Как организованы материалы?

В папке [materials](materials) содержатся подпапки, пронумерованные по семестрам. Нумерация, преимущественно, актуальна для тех, кто учится на направлении 24.05.07.

На данный момент присутствует большинство материалов третьего семестра и далее. Планируется переписывание старых бумажных конспектов с первого курса, благодаря чему будет собрана почти полная коллекция предметов.

В папках с каждым предметом лежит единый исходник на Typst и/или исходники в Markdown по каждой отдельной паре. Исходники рисунков хранятся в папках source-figures.
