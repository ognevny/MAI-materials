#### Главные оси инерции

Главными осями инерции называются оси, для которых смешанные моменты инерции в индексе которых присутствует название этой оси равны 0 (J<sub>xy</sub> = J<sub>xz</sub> = 0 => Ox - главная ось инерции). Можно доказать, что в любой точке твердого тела любой формы можно построить по крайней мере 3 взаимноортоганальные главные оси инерции.

Если Ox, Oy, Oz - главные оси инерции, то <a>K&#773;<sub>0</sub> = (J<sub>x</sub>&omega;<sub>x</sub>, J<sub>y</sub>&omega;<sub>y</sub>, J<sub>z</sub>&omega;<sub>z</sub>)</a>.
