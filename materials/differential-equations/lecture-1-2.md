### ДУ 1 порядка

Уравнение, где искомая функция находится под знаком производной или под знаком дифференциала, называется дифференциальным. Уравнение, связывающее независимую x, искомую y(x) и y'(x), называется ДУ 1 порядка.

а) y' = f(x, y)

б) F(x, y, y') = 0 - неявно выраженная

в) P(x, y)dx + Q(x, y)dy = 0

<u>y' = dy/dx</u>

y' + y = 1 => y' = 1 - y; y' + y - 1 = 0; dy/dx + y = 1 => dy/dx = 1 - y => (1 - y)dx - dy = 0

Фунция y = <a>&phi;</a>(x), непрерывно диффернцируемая на интервале (a; b) называется решением ДУ, если подстановка <a>&phi;</a>(x), <a>&phi;'</a>(x) обращает ДУ в тождество для любого x из (a; b).

1) <a>&phi;</a>(x) = 1; <a>&phi;'</a>(x) = 0 => ДУ: 0 + 1 <a>&equiv;</a> 1 => y = 1 - решение ДУ.
2) y = <a>&phi;</a>(x) = x<sup>2</sup>; y' = <a>&phi;'</a>(x) = 2x => ДУ: 2x + x<sup>2</sup> <a>&nequiv;</a> 1 => y = x<sup>2</sup> не является решением ДУ.

3) y = <a>&phi;</a>(x) = Ce<sup>-x</sup> + 1; y' = <a>&phi;'</a>(x) = -Ce<sup>-x</sup> => ДУ: -Ce<sup>-x</sup> + 1 - -Ce<sup>-x</sup> <a>&equiv;</a> 1 => y = -Ce<sup>-x</sup> + 1 - решение ДУ

#### Задача Коши

Поиск решения ДУ y' = f(x, y), удовлетворающего дополнительному условию y(x<sub>0</sub>) = y<sub>0</sub> (начальные условия - НУ), называется задачей Коши.

y' = 2x => y(x) = <a>&int;</a>2xdx = x<sup>2</sup> + C - семейство интегральных кривых.

y = <a>&phi;</a>(x) = x<sup>2</sup> + C; y' = <a>&phi;'</a>(x) => 2x + 0 <a>&equiv;</a> = 2x

y(1) = 2, y(1) = x<sup>2</sup> + C|<sub>x=1</sub> = 2

1<sup>2</sup> + C = 2 => C = 1 => y = x<sup>2</sup> + 1 - решение задачи Коши

y = Ce<sup>x</sup>, y' = Ce<sup>x</sup> => ДУ: <a>Ce<sup>x</sup> &equiv; Ce<sup>x</sup></a>

НУ y'(0) = -1; y(0) = Ce<sup>x</sup>|<sub>0</sub> = Ce<sup>0</sup> = -1 => C = -1 => y = -e<sup>x</sup> - решение задачи Коши
