## Лекции 1-3

### Случайные события

#### Основные понятие теории вероятностей

Эксперимент (опыт) состоит в том, что проводится испытание при выполнении некоторого комплекса условий, либо создаваемых искусственно, либо осуществляемых не по воле экспериментатора. Должны быть указаны условия и наблюдаемый результат.

Событие - результат (исход) события

Эксперименты - детерменированные или случайные (стохастические)

Случайный эксперимент - эксперимент, проводимый в условиях стохастической однородности (воспроизводимость, непредсказуемость, устойчивость частоты события)

Случайное событие (СС) - событие, которое при осуществлении эксперимента может произойти или не произойти.

##### Виды событий

Пространство элементарных событий <a>&Omega;</a> - совокупность всех взаимоисключающих событий.

Событие - любое подмножество пространства элементарных событий.

Элементарное событие <a>&omega;</a> - любой простейший неделимый исход эксперимента.

Совместные - появление одного события не исключают появление другого.

Противоположные события - если в одном и том же эксперименте они несовместны, и одно из них обязательно произойдет.

Достоверное событие - данном эксперименте обязательно произойдет.

Невозможное событие - обязательно не произойдет.

Благоприятное событие - если появление его способствует появлению другого события.

Равные события - содержат одни и те же элементы.

#### Алгебра событий

Сложение, умножение, разность (\\), симметрическая разность (<a>&oplus;</a>).

A(B + C) = AB + AC; (A + B)(A + C) дистрибутивность

#### Вероятность события

##### Классическое определение вероятности (КОВ) и его свойства

1. Полная группа событий образует совокупность событий A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>n</sub> => <a>&Sigma;A<sub>i</sub> = &Omega;; A<sub>i</sub>A<sub>j</sub> = &empty; &forall;i &ne; j</a>
2. События называются равновозможными, если в результате эксперимента появляются одинаково часто

P(A) = m/n; P(<a>&Omega;</a>) = 1; P(<a>&empty;</a>) = 0; 0 < P(A) < 1 (A <a>&ne; &Omega;, A &ne; &empty;) => 0 &le; P(A) &le; 1</a>.

###### Недостатки КОВ

Не всегда можно представить результаты как совокупность равновозможных событий; m и n конечные, что не всегда так; слова "равновозможность" и "равновероятность" - синонимы.

##### Геометрическое определение вероятности

P(A) = *mes q* / *mes G* - мера области, благоприятной появлению события A к мере всей области.

<img src=source-figures/lect1-3-2.png />

##### Статистическое определение вероятности

Событие А появилось m раз за n опытов, тогда m - абсолютная частота, W(A) = m/n - относительная частота.

<img src=source-figures/lect1-3-3.png />

P(A) - число, около которого группируется значение частот при большом количестве опытов.

##### Аксиоматическое построение теории вероятностей

Это осуществили математики Бильберт (1899 год) и Колмогоров (1933 год).

Алгебра событий <a>&#8497;</a> называется любое непустое множество, удовлетворяющее следующим требованиям:

<a>A &isin; &#8497; => A&#772; &isin; &#8497;</a>

<a>A, B &isin; &#8497; => A &#8899; B &isin; &#8497;, A &#8898; B &isin; &#8497;</a>

<a>&#8497;<sub>0</sub> = {&empty;, &Omega;}</a>

<a>&#8497; = {&empty;, A, A&#772;, &Omega;}</a>

**<a>&sigma;</a>-алгебра (F)**

<a>&Omega; &isin; F</a>

<a>A &isin; F => A&#772; &isin; F</a>

<a>A<sub>1</sub>, A<sub>2</sub>, ... &isin; F => &Sigma;A<sub>i</sub> &isin; F, &Pi;A<sub>i</sub> &isin; F</a>

Вероятностной мерой называется числовая функция, заданная в алгебре событий.

F: <a>A &isin; F</a>; P(A)

<a>P(A) &ge; 0 &forall;A &isin; F</a>

P(<a>&Omega;</a>) = 1

<a>P(&Sigma;A<sub>i</sub>) = &Sigma;P(A<sub>i</sub>), A<sub>i</sub>A<sub>j</sub> = &empty; &forall;i &ne; j &forall;A<sub>i</sub> &isin; F</a>

<a>lim(P(A<sub>n</sub>)) = 0, A<sub>1</sub> &sup; A<sub>2</sub> &sup; ... &sup; A<sub>n</sub> &sup; ... => &Pi;A<sub>i</sub> = &empty; &forall;A<sub>i</sub> &isin; F</a>

Конечно аддитивная вероятность, заданная на <a>&sigma;</a>-алгебре называется непрерывной, если <a>&forall;</a> убывающей последовательности, имеющей пустое пересечение имеет место равенство:

<a>lim(P(A<sub>n</sub>)) = 0 &lt;= [P(A<sub>n</sub>), F]; A<sub>1</sub> &sup; A<sub>2</sub> &sup; ... &sup; A<sub>n</sub> &sup; ... && &Pi;A<sub>i</sub> = &empty;</a>

**Свойства <a>&sigma;</a>-алгебры**

1. P(<a>&empty;</a>) = 0
2. P(<a>A&#773;</a>) = 1 - P(A)
3. <a>P(A) &le; P(B), A &sub; B</a>
4. <a>0 &le; P(A) &le; 1</a>
5. P(A+B) = P(A) + P(B) - P(AB)
6. P(A+B) <a>&le;</a> P(A) + P(B)
7. <a>P(&Sigma;A<sub>i</sub>) = &Sigma;P(A<sub>i</sub>) - &Sigma;<sub>&le;i&lt;j&le;n</sub>P(A<sub>i</sub>A<sub>j</sub>) + &Sigma;<sub>1&le;i&lt;j&lt;m&le;n</sub>P(A<sub>i</sub>A<sub>j</sub>A<sub>m</sub>) - ... - (-1)<sup>n</sup>P(&Pi;A<sub>i</sub>)</a>
8. <a>P(&Sigma;A<sub>i</sub>) &le; &Sigma;P(A<sub>i</sub>)</a>

#### Основные формулы вычисления вероятностей

Для несовместных событий (P(AB) = 0) P(A+B) = P(A) + P(B); <a>P(&Sigma;A<sub>i</sub>) = &Sigma;P(A<sub>i</sub>) => &Sigma;P(A<sub>i</sub>) = 1, P(A) + P(A&#773;) = 1 для &Sigma;A<sub>i</sub> = &Omega;</a>.

При наличии совместности P(A+B) = P(A) + P(B) - P(AB); <a>P(&Sigma;A<sub>i</sub>) = &Sigma;P(A<sub>i</sub>) - &Sigma;<sub>1&le;i&lt;j&le;n</sub>P(A<sub>i</sub>A<sub>j</sub>) + &Sigma;<sub>1&le;i&lt;j&lt;m&le;n</sub>P(A<sub>i</sub>A<sub>j</sub>A<sub>m</sub>) - ... - (-1)<sup>n</sup>P(&Pi;A<sub>i</sub>)</a>.

##### Пример

Всего имеется 30 жетонов с номерами от 1 до 30. Событие A - номер делится на 2, событие B - номер делится на 3. P(A+B) = 15/30 + 10/30 - 5/30 = 2/3.

#### Условная вероятность и независимость события

Два события называются независимыми, если вероятность появления одного события не зависит от того, появилось ли другое событие или нет.

P(B/A) = P(AB)/P(A), при P(A) = 0 P(B/A) не определена.

Условной вероятностью события называется вероятность этого события при условии, что другое произошло.

1. <a>0 &le; P(A/B) &le; 1</a>
2. P(<a>A&#773;/B</a>) = 1 - P(A/B)
3. P(A/A) = 1
4. P(A<sub>1</sub>+A<sub>2</sub> / B) = P(A<sub>1</sub>/B) + P(A<sub>2</sub>/B) - P(A<sub>1</sub>A<sub>2</sub>/B)
5. AB = <a>&empty;</a> => P(A/B) = 0
6. A <a>&sub;</a> B => P(B/A) = 1

##### Теорема умножения вероятностей

P(AB) = P(A)P(B/A) = P(B)P(A/B) - ЗС

P(A<sub>1</sub>A<sub>2</sub>A<sub>3</sub>...A<sub>n</sub>) = P(A<sub>1</sub>)P(A<sub>2</sub>/A<sub>1</sub>)P(A<sub>3</sub>/A<sub>1</sub>A<sub>2</sub>)...P(A<sub>n</sub>/A<sub>1</sub>A<sub>2</sub>...A<sub>n-1</sub>)

P(AB) = P(A)P(B) - НЗС (P(B/A) = P(B))

<a>P(A<sub>1</sub>A<sub>2</sub>A<sub>3</sub>...A<sub>n</sub>) = &Pi;P(A<sub>i</sub>)</a>

##### Формулы полной вероятности и Байеса

H<sub>1</sub>, H<sub>2</sub>, ..., H<sub>n</sub> - полная группа событий, образованная несовместными событиями, то есть H<sub>i</sub>H<sub>j</sub> = <a>&empty; &forall;i&ne;j</a> и <a>&Sigma;H<sub>i</sub> = &Omega;</a>.

Следствием теоремы сложения вероятностей для несовместных событий и умножения для независимых событий является формула полной вероятности:

P(A) = <a>&Sigma;P(H<sub>i</sub>)P(A/H<sub>i</sub>)</a>

<img src=source-figures/lect1-3-5.png />

Формула Байеса: P(H<sub>i</sub>/A) = P(H<sub>i</sub>)P(A/H<sub>i</sub>) / P(A) - выведено Лапласом
